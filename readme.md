## [b00f.github.io](http://b00f.github.io/)

### Install

First [install](https://jekyllrb.com/docs/installation/ubuntu/) Jekyll.

```
gem install jekyll bundler
bundle install
bundle exec jekyll serve
```

### Useful commands

- Use Prettier to format markdown pages:

```
yarn run prettier
```

- Exiftool on images

```
yarn run exif
```
